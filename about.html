Mask Detection

in Practice

mask detection technology is trained to detect all the variations of face masks that are available. From medical masks to personalized cloth masks, the system will accurately detect and confirm mask compliance. I have used Python, openCV and google collab for this project.Trained my computer with mask-dataset including all mask faceed people and unmasked people and run the code regarding this on my google collab. Retail

Retailers need to monitor their premises to control the current occupancy and wearing of masks. Digital screens can be used to display information for both the number of people allowed in the store and mask detection. Learn more about our occupancy monitoring system

Airport

This face mask detector can be very effectively used at airports mainly for entrance flow management and monitoring. The software can be added to any access gate or entrance to make sure that all passengers follow the safety rules when boarding a plane.

Public Transport

Wearing face masks in public transport will be mandatory in many parts of the world. Public transport organizations can use the software to automate the checking process with very little resources needed and with efficiency.

Coorperate Buildings

With many office buildings opening up and employees coming back to work, face mask detection can be used to maintain a safe environment for everyone. The mask detection system can send alerts or reminders to those not complying.
